## 画面詳細

### S-1: ログイン

- メールアドレス入力欄
- パスワード入力欄
- ログインボタン
- エラーメッセージ表示エリア

### S-2: 打刻（ホーム）

- 現在時刻の表示
- 本日の打刻状態（未出勤 / 出勤済み / 退勤済み）
- 出勤ボタン（未出勤時のみ活性）
- 退勤ボタン（出勤済み・未退勤時のみ活性）
- 本日の出勤時刻・退勤時刻の表示

### S-3: 勤怠履歴（自分）

- 年月選択（デフォルト: 当月）
- 日別一覧テーブル
  - 日付 / 出勤時刻 / 退勤時刻 / 勤務時間 / 残業時間
  - 各行に「修正」リンク
- 月次集計エリア
  - 出勤日数 / 合計勤務時間 / 合計残業時間

### S-4: 打刻修正

- 対象日の表示
- 出勤時刻入力欄（datetime）
- 退勤時刻入力欄（datetime、任意）
- 保存ボタン / キャンセルボタン

### S-5: 社員勤怠一覧（上司・管理者）

- 社員選択ドロップダウン（上司: 部下のみ表示 / 管理者: 全社員）
- 年月選択
- 日別一覧テーブル（S-3 と同構成）
- 各行に「修正」リンク（管理者のみ、打刻修正 → S-4）
- 月次集計エリア

### S-6: 社員管理一覧

- 社員テーブル
  - 社員コード / 氏名 / メールアドレス / ロール / 状態
  - 各行に「編集」「削除」ボタン
- 「新規登録」ボタン

### S-7: 社員登録

- 氏名入力欄
- メールアドレス入力欄
- パスワード入力欄
- ロール選択（一般社員 / 管理者）
- 登録ボタン / キャンセルボタン

### S-8: 社員編集

- 氏名入力欄（既存値プリセット）
- メールアドレス入力欄
- ロール選択
- 保存ボタン / キャンセルボタン
